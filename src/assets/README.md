# assets/

Static assets for SnapStore Studio.

## Directory Structure

```
assets/
├── images/       — Static images (logos, illustrations, placeholders)
├── icons/        — SVG icon files (device frames, brand icons)
└── fonts/        — Self-hosted font files (if not using Google Fonts)
```

> Note: Images used in the canvas or editor should be uploaded via the
> Storage service, not placed here. This directory is for app-level static assets only.
